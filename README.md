# MeetShop（觅店）本地生活服务平台

## 📌 项目简介

**MeetShop（觅店）** 是一个基于 Spring Boot 的本地生活服务平台，面向城市用户的到店消费场景，提供商户信息展示、优惠券发放与秒杀、用户互动与社交等核心功能。
项目采用前后端分离架构，通过 Redis 与数据库集群支撑高并发访问，重点解决了缓存一致性、秒杀超卖、分布式锁以及异步下单等典型高并发问题。

该项目从真实业务场景出发进行设计，适用于学习和实践 **Redis 高并发优化、分布式系统设计以及秒杀场景解决方案**。

## 🚀 核心技术亮点

### 1️⃣ 分布式登录与权限控制

* 基于 **Redis + Token** 实现分布式 Session 共享
* 使用 **拦截器** 完成登录校验与用户信息刷新

### 2️⃣ 缓存设计与一致性保障

* 采用 **Cache Aside 模式** 保证数据库与缓存的一致性
* 对高频访问数据进行缓存，显著降低数据库压力
* 针对缓存穿透、缓存击穿、缓存雪崩问题，采用：

    * 空值缓存
    * 互斥锁
    * 逻辑过期等方案进行优化

### 3️⃣ 秒杀系统设计

* 使用 **Redis + Lua 脚本** 对用户秒杀资格进行预校验，保证原子性
* 结合 **乐观锁** 解决库存超卖问题
* 基于 **Redis 分布式锁** 实现集群环境下的“一人一单”控制

### 4️⃣ 异步下单与系统解耦

* 基于 **Redis Stream** 构建消息队列
* 实现秒杀下单的异步化处理，提高系统吞吐量与稳定性

### 5️⃣ 社交与排行榜功能

* 使用 **ZSet** 实现点赞排行榜
* 使用 **Set** 实现关注与共同关注功能
